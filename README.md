Aplicación Full Stack diseñada para simplificar la administración de personal dentro de una organización. Permite gestionar empleados, departamentos y sueldos de manera centralizada y eficiente.

---

## 🚀 Funcionalidades Principales
* **CRUD de Empleados:** Alta, baja, modificación y visualización de datos del personal.
* **Gestión de Departamentos:** Organización jerárquica de la empresa.
* **Cálculo de Nómina:** Gestión de sueldos y detalles salariales.
* **Interfaz Dinámica:** Frontend reactivo que consume la API REST de forma fluida.

## 🛠️ Stack Tecnológico
* **Backend:** Java 17, Spring Boot 3, Spring Data JPA.
* **Frontend:** React, JavaScript (ES6+), CSS3.
* **Base de Datos:** MySQL / MariaDB.
* **API:** RESTful con arquitectura de capas (Controller, Service, Repository).

---

## ⚙️ Instalación y Uso
1. **Backend:** Configurar el archivo `application.properties` con tus credenciales de MySQL y ejecutar la aplicación Spring Boot.
2. **Frontend:** Navegar a la carpeta de la app de React y ejecutar:
